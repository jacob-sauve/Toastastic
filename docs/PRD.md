# Project Requirements Document

## Description
The Toastastic project aims to:
- design a board
- produce code
In order to be able to read and write to our (dumb) toaster's state, i.e. toast bread and know when it is ready.
This will be done through a self-hosted web portal.

## Requirements
1. Be able to actuate the toaster's main lever
2. Be able to adjust toasting "strength" via the knob
3. Have an emergency stop
4. Program a toast order to be ready for a given time
5. Automatic bread magazine
6. 

## Constraints
1. Bread remains intact to the same degree as manual toasting (no holes, etc.)
2. Low noise (RIP Martin)
3. No moving the toaster itself
4. No open-air loaves
